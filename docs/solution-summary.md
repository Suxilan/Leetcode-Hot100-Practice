# 解法归纳（第一组）

## LC206 链表反转

- 题型: 链表指针操作
- 核心思路: prev, cur, nxt 三指针原地反转
- 关键步骤:
  1. 保存 nxt = cur.next
  2. cur.next = prev
  3. prev = cur, cur = nxt
- 易错点: 未提前保存 nxt 会导致链断
- 复杂度: 时间 O(n), 额外空间 O(1)

## LC283 移动零

- 题型: 双指针（快慢指针）
- 核心思路: slow 指向下一个非零该放置的位置，fast 扫描数组
- 关键步骤:
  1. 遍历 fast
  2. 若 nums[fast] 非零，交换 nums[slow], nums[fast]
  3. slow += 1
- 易错点: 不能破坏非零元素相对顺序
- 复杂度: 时间 O(n), 额外空间 O(1)

## LC3 无重复字符的最长子串

- 题型: 滑动窗口 + 哈希表
- 核心思路: 维护无重复窗口 [left, right]
- 关键步骤:
  1. 哈希记录字符最近出现位置
  2. 遇到重复字符时移动 left
  3. 持续更新答案长度
- 易错点: left 只能右移，不能回退
- 复杂度: 时间 O(n), 额外空间 O(k)
